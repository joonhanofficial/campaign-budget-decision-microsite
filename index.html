Problem

Campaign reporting should not stop at metric snapshots.

Spend, revenue, conversions, CPA, ROAS, and profit are useful only when they help a marketing team decide what to do next. The core question was not simply which campaigns performed well. It was which campaigns should receive more budget, which should be reviewed, which should hold, and which should be reduced before they create more risk.

Method

From performance metrics to budget action logic.

The analysis translated core campaign indicators into a decision layer: ROAS, CPA, profit, and conversion rate informed the final budget action.

Input

ROAS

Measures revenue return against spend to identify scaling potential.

High ROAS can signal scaling potential, but it does not automatically mean the campaign is low risk.
Input

CPA

Shows how expensive each conversion is and where efficiency risk appears.

CPA helps identify efficiency pressure. A campaign can generate conversions but still be too expensive to scale.
Input

Profit

Separates surface-level volume from financially meaningful outcomes.

Profit separates surface-level activity from financially meaningful outcomes.
Input

Conversion Rate

Connects traffic quality to conversion behavior before budget decisions.

Conversion rate helps check whether traffic quality is strong enough to support a budget decision.
Output

Budget Action

Increase Budget, Review, Hold, or Reduce Budget.

Budget Action Breakdown

Every campaign receives a next-step recommendation.

The output was designed for budget review meetings, not passive reporting.

Business Implication

The practical output is a budget review workflow.

A marketing team can use this analysis to prioritize where budget should move next. Increase Budget campaigns become scaling candidates, Review campaigns become discussion points, Hold campaigns remain monitored, and Reduce Budget campaigns become risk controls. The value is not another dashboard view. The value is a structured decision layer that helps teams spend with more confidence.
