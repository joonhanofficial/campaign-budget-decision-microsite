# Campaign Budget Decision Microsite

A one-page static microsite for the analytics case study **"Turning Campaign Metrics Into Budget Decisions."**

The page presents how Python and Excel were used to analyze 10,000 campaign rows across spend, revenue, conversions, CPA, ROAS, profit, and platform, then translate those metrics into budget review actions.

## Files

- `index.html` - Page structure and case study content
- `style.css` - Responsive dark analytics command-center design
- `script.js` - Smooth scrolling, scroll reveals, and animated number counters
- `README.md` - Project overview and preview instructions

## Preview Locally

Open `index.html` directly in a browser, or run a simple local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## GitHub Pages

This project is GitHub Pages compatible because it uses plain HTML, CSS, and JavaScript with no build step.

To deploy from GitHub:

1. Go to the repository settings.
2. Open **Pages**.
3. Set the source to the `main` branch.
4. Select the repository root as the publishing folder.
